Netflix Data Analysis and Visualization Project¶

Introduction¶

Netflix has a massive and diverse content library, spanning movies, TV shows, documentaries, and more.
This project analyzes Netflix’s dataset, cleans and preprocesses it, creates meaningful features, and performs advanced visualizations.
Additionally, it includes a simple recommendation system to suggest similar content based on genres.

Project Overview¶

This project focuses on cleaning, analyzing, and visualizing Netflix data to understand content trends, explore relationships, and build a basic recommendation system.
The dataset contains movies and TV shows, along with their release year, ratings, duration, cast, and more.

Key Objectives:

  • Clean and prepare the dataset for analysis
  • Explore content trends across time, genres, and countries
  • Visualize patterns using advanced charts
  • Create a simple content-based recommendation system

from sklearn.feature_extraction.text import CountVectorizer
from sklearn.metrics.pairwise import cosine_similarity

# Fill missing genre values
df['listed_in'] = df['listed_in'].fillna('')

# Vectorize genres
vectorizer = CountVectorizer(stop_words='english')
genre_matrix = vectorizer.fit_transform(df['listed_in'])

# Compute similarity
cosine_sim = cosine_similarity(genre_matrix, genre_matrix)

# Index map for title
indices = pd.Series(df.index, index=df['title']).drop_duplicates()

# Recommendation function
def recommend(title, cosine_sim=cosine_sim):
    if title not in indices:
        return f"'{title}' not found in dataset."
    idx = indices[title]
    sim_scores = list(enumerate(cosine_sim[idx]))
    sim_scores = sorted(sim_scores, key=lambda x: x[1], reverse=True)
    sim_scores = sim_scores[1:6]
    return df['title'].iloc[[i[0] for i in sim_scores]]

Recommendation System¶

Method:

  • Used CountVectorizer to encode genres into a matrix.
  • Calculated cosine similarity between all titles.
  • Created a function recommend(title) to suggest top 5 similar titles.
